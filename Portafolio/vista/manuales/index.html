Objetivos de este Manual

Los objetivos que tiene este manual, son el de mostrar cómo se debe realizar un manual de usuarios y un manual de clientes con su debido paso a paso.


Antes de empezar, hay que aclarar que el significado de la palabra "Manual" se puede definir como un libro o folleto, en donde se dan las instrucciones para el uso o mantenimiento de un aparato, un sistema o entre otros. Claro ejemplo podría ser el manual de usuario para un software.
