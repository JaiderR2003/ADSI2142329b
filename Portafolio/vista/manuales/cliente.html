¿Que es un manual técnico?


Un manual técnico es aquel que va dirigido a un público con conocimientos técnicos sobre algún área.
La documentación de proyectos es importante para identificar mas fácilmente los aspectos y características que forman parte de un proyecto



Paso

Objetivo general y especifico


En este apartado, se debe describir con mucha claridad el objetivo del sistema y los objetivos que se cumplieron con el desarrollo del sistema



Paso

Contenido técnido


El contenido técnico, es toda información textual o gráfica que describe o da instrucciones de un producto
En estos, se suele colocar información acerca de:
Deescripcion de los procesos: Se describen los procesos a los cuales va a apoyar el sistema de información
Diagramas UML: Se presentan los casos de uso, clases y otros que hallan sido desarrollados para el programa
Descripción de campos requeridos por pantalla: Se explican los campos que se utilizan en cada formulario
Modelo Lógico de datos: Se uncluye el diagrama Entidad relación y el diccionario de datos



Paso

Plataforma del usuario

En este apartado se deben mencionar las diversas herramientas que se usaron para el desarrollo del sistema. Por ejemplo el lenguaje en que fue escrito, la base de datosque se usó entre otros



Paso

Interfaz grafica

Se representa todo lo referente a la vista. Se debe incluir el mapa de navegacion y la descripción paso a paso
