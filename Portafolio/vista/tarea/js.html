Cómo se trabaja con las funciones flecha

La funcion flecha, es un modo para definir funciones con una sintaxis más abreviada. Esto nos permitirá ahorrar tiempo a la hora de programar

A continuación veremos como hacer un código que nos elelve al cuadrado un numero

function cuadrado (num){
return num*num;
}
                    
console.log('tradicional: '+cuadrado(10))

Y ahora vamos a realizar el mismo ejercicio, pero con la función flecha

var r=nume=>nume*nume;

console.log('flecha: '+r(5))

Como podemos observar, la función flecha es mucho más corta que la anterior y tiene un igual y mayor que (=>) que nos ayuda a identificar que una función flecha



Que son las fracciones?

Las fracciones son una unidad dividida, esto quiere decir que es un pedazo que formaba parte algo. Por ejemplo: un pedazo de un pastel es una fraccion

Qué es la herencia?

La herencia permite que una clase tenga el mismo comportamiento que otra clase y amplíe o adapte ese comportamiento para brindar una acción especial para necesidades específicas

                    class ZonasVerdes{
                        constructor(IdZona, Nombre){
                            this._IdZona=IdZona;
                            this._Nombre=Nombre;
                        }
                    
                        set IdZona(IdZona){
                            this._IdZona=IdZona;
                        }
                    
                        get IdZona(){
                            return this._IdZona;
                        }
                    
                        set Nombre(Nombre){
                            this._Nombre=Nombre;
                        }
                    
                        get Nombre(){
                            return this._Nombre;
                        }
                    }
                    
                    class ZonasNaturales extends ZonasVerdes{
                        constructor (IdZona, Nombre, Ubicacion, Descripcion, Imagen){
                            super(IdZona, Nombre);
                            this._Ubicacion=Ubicacion;
                            this._Descripcion=Descripcion;
                            this._Imagen=Imagen;
                        }
                    
                        set Ubicacion(Ubicacion){
                            this._Ubicacion=Ubicacion;
                        }
                    
                        get Ubicacion(){
                            return this._Ubicacion;
                        }
                    
                        set Descripcion(Descripcion){
                            this._Descripcion=Descripcion;
                        }
                    
                        get Descripcion(){
                            return this._Descripcion;
                        }
                    
                        set Imagen(Imagen){
                            this._Imagen=Imagen;
                        }
                    
                        get Imagen(){
                            return this._Imagen;
                        }
                    }

La palabra "extends" nos permite heredarle a la clase "hija" atributos de la superclase, ademas de que con solo ver la palabra "extends" ya podemos deducir que es una herencia
